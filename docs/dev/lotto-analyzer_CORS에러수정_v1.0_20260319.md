## 개요
- 최신 로또 데이터 업데이트 기능이 `corsproxy.io` 등 웹 프록시의 접속 불량/과금화로 인해 동작하지 않는 문제 수정.

## 세부 수정 기획
1. **프록시 리스트 변경**
   - 안정적인 퍼블릭 CORS 프록시(`api.allorigins.win/get`, `api.codetabs.com/v1/proxy`) 로 교체하여, `index.html`의 하드코딩된 API 경로 수정.
2. **응답 데이터 처리 개선**
   - 프록시 종류에 따라 응답 형태(순수 JSON, 문자열 래핑 등)를 유연하게 파싱하도록 `fetchDraw` 함수 부분의 응답처리 로직 개선. JSON 포맷이 이중 직렬화된 경우의 처리 로직을 추가합니다.
