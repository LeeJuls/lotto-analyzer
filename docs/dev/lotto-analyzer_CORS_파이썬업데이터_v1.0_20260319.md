# 개발 명세서
- 프로젝트명: lotto-analyzer
- 문서명: CORS_파이썬업데이터_명세서
- 버전: v1.0
- 작성일: 2026-03-19

## 개요
동행복권 API의 외부 프록시 차단 이슈로 인해 프론트엔드 종속적인 CORS 우회 방식이 모두 실패함에 따라, 사용자 로컬 환경(파이썬)에서 직접 데이터를 받아와 `index.html` 파일을 갱신하는 방향으로 전환함.

## 세부 수정 기획
1. **`update_lotto.py` 작성**
   - 역할: 동행복권 API 호출 후 누락된 당첨 회차 데이터를 수집함.
   - 동작 로직: `index.html` 파일을 열어 `EMBEDDED_RAW` 배열 변수를 정규표현식으로 찾고 마지막 회차 정보를 통해 누락된 회차를 확인하여 파싱 후 JSON 형태로 변환, 파일 텍스트를 대체하여 저장함.
2. **`index.html` 변경**
   - 상단의 "데이터 갱신" 기능이 브라우저에서 막히므로, `python update_lotto.py` 를 실행하라는 알림이나 레이블 안내로 변경함.
