- 버전: v1.0
- 변경일: 2026-03-19

## 변경 사항 및 의사결정 내역
1. **문제 상황 지속**
   - 동행복권 API 자체에 강력한 Cloudflare WAF 봇 방지가 걸려 있어 프록시 우회나 일반 파이썬 `urllib`/`requests` 접근조차 막히는 상황이 확인됨.
   - 그래서 1213회차 등 최신 데이터를 가져올 수 없었음.
2. **의사 결정: 네이버 검색 결과 크롤링으로 우회**
   - 브라우저 제어나 수동 다운로드보다 빠르고 명확한 우회를 위해 "1213회로또" 등 네이버 검색 결과를 `BeautifulSoup`으로 직접 크롤링하는 `update_lotto.py` 스크립트로 전면 수정.
3. **작업 및 개선 내역**
   - `update_lotto.py` 에 네이버 검색 HTML 페이지를 파싱하고 `[ span.ball ]` 태그를 직접 읽어 들여 당첨 번호를 가져오는 로직 추가 (`fetch_lotto_naver` 함수 구현).
   - 정상적으로 우회하여 1213회, 1214회, 1215회 당첨 번호까지 깔끔하게 파싱 후 `index.html` 파일을 로컬에 자동 병합 저장하도록 완료.
   - 브라우저 상단의 "데이터 갱신" 버튼은 폴더의 파이썬 스크립트를 수동으로 돌리도록 안내 문구로 연결.
